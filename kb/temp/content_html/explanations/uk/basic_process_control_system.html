Базова система управління процесами (BPCS) - це контрольно-вимірювальні прилади та системи, які встановлюються для моніторингу та контролю нормальних виробничих операцій за допомогою, але не обмежуючись комбінаціями одноконтурних пневматичних та електронних моніторів та контролерів, програмовані логічні контролери та розподілені системи управління.
(a) BPCS необхідний для роботи установки або процесу.
